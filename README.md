# 👟 KICKS — E-Commerce Web Application

**KICKS** is a modern, responsive **eCommerce web application** built with **Next.js**, **Tailwind CSS**  
The project focuses on a clean shopping experience with a professional checkout flow inspired by real-world brands. 
## Live Link :https://online-shop-red.vercel.app/
---

## ✨ Features

- 📱 Fully **responsive UI** (mobile, tablet, desktop)
- 🛍️ Shopping cart & checkout page
- 👟 Product details (image, size, quantity)
- 💰 Order summary with pricing breakdown
- ⚡ Optimized images using `next/image`
- 🎨 Modern, minimal eCommerce design
- ♻️ Reusable UI components

---

## 🧰 Tech Stack

- **Next.js (App Router)**
- **React**
- **Tailwind CSS**
- **TypeScript** (optional but recommended)

🚀 Getting Started
1️⃣ Clone the repository
git clone https://github.com/sadman1075/online-shop
cd kicks
2️⃣ Install dependencies
npm install
# or
yarn install

4️⃣ Run the development server
npm run dev

Open 👉 http://localhost:3000
